## The Plan
My vision is to create a made-up Canvas plugin called "Atomic Pokedex".
It will enable the user to learn more about Pokemon, and even take quizzes to test their knowledge.

We will use a screenshot to make it look like it's in Canvas, and the interactive part is the actual Pokedex.
I set up this Vue + Vite project, and we'll use this to run the project locally.
I will provide the screenshot of Canvas (src/assets/CanvasBackground.png) as well as the APIs to use to fetch the data (for now, we'll just use GET https://pokeapi.co/api/v2/pokemon/{id or name}/), but I could use help making the structure of the project because I only have less than 3 hours to make it.

### Layout
The layout of the app will be like this:

The information displayed for a single Pokemon will include:
- A Pokedex vue component
    - A Start quiz button that opens up the quiz panel
    - A display vue component (this will either prompt the user to search and select a Pokemon, or display the image of the selected pokemon)
    - A search bar (filters the Pokemon by name)
    - A results vue (displays either cards that each have the Pokemon name and picture if in Card view, or simply a list of names if in List view)
    - A set of radio buttons to change the view mode between Card and List
- A QuizPanel vue component (Will be added as a stage 2 feature, so we can ensure the Pokedex vue component works completely first)
